2. Open a command prompt(cmd) window and set the directory address to the current folder.
3. Compile the Server2023.java program using javac Server.java command.
4. Run the Server program using java Server command.
5. Open a new command prompt(cmd) and set directory to the current folder.
6. Compile the Client.java program using javac Client.java command.
7. Run the Client program using java Client command.
